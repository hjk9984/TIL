# Algorithm
## cpp condition

if문의 조건식에서 순서대로 확인한다.

```cpp
    stack<int> tmp;
    if(tmp.top() && !tmp.empty()) cout << "ok";     //컴파일 에러 발생
    if(!tmp.empty() && tmp.top()) cout << "ok";     //통과
```

    vector<int> v;
