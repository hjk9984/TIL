# OS
운영체제란 컴퓨터의 자원 즉, 하드 웨어를 총괄하고 어플리케이션을 돌일 수 있도록 관리해 주는 SW

우리가 보통 하용하는 리눅스인 우분투, centOS는 리눅스의 **배포판**이라고 한다.
<br>

# 커널
커널은 사물의 중심이란 뜻을 가지며 운영체제의 중심을 의미한다.<br>
리눅스의 루트경로에 있는 'vmlinuz' 파일이나 'boot' 디렉터리에 있는 파일들이 커널의 본체다.
<br>

# 디바이스
커널의 중요한 역할 중 하나가 하드웨어를 관리하는 것이다. <br>
예를 들어 여러 종류의 CPU가 있고 이를 조작하게 해주는 SW를 **디바이스 드라이버**라 한다.
<br>

# 시스템 콜
하드웨어를 직접 다루는 건 커널만이 할 수 있기에 일반 프로그램이 커널에 디바이스 조작 작업을 의뢰하기 위해 사용하는 것이 **시스템 콜**이라고 한다. 아래는 시스템 콜의 종류다.
* open, read, write, fork, exec, stat, unlink
<br>

# 라이브러리 함수
라이브러리 함수는 내부적으로 시스템 콜을 사용하기도 하고 그렇지 않을 수도 있다. printf()의 경우 내부에서 write()라는 시스템 콜을 사용하고 strlen()의 경우는 어떤 시스템 콜도 사용하지 않는다.
